# Floorball Shot Plotter

A Python-based desktop tool for tracking, analyzing, and visualizing floorball shots against your team, including video-assisted plotting, danger-zone overlays, heatmaps, and xG-ready event logging.

Current app version: `0.3.0-beta`

This project is still pre-1.0. Version `0.x` means active development / beta-quality rather than a finished stable release. The old `v3.0` label was only a development-phase name and should not be treated as production readiness.

## Features

- Opponent shot & goal tracking
- Pass-origin logging
- Video-assisted plotting with calibration
- Shot Zone / danger-zone overlay
- Interactive shot map & heatmaps
- Shot Log filtering and column visibility
- Match save/load and CSV export
