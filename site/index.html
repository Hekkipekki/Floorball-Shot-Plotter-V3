How to install

  1. Click Download latest beta.
  2. Unzip the downloaded Windows package.
  3. Open the folder and run Floorball-Shot-Plotter.exe.
  4. If Windows SmartScreen warns you, choose More info and then Run anyway only if you trust the source.
  5. Test with demo shots or your own match data.
