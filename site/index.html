What testers can try

  • Heatmap and goal/save heatmap views
  • Match save/load and CSV export
  • Video linking and segment playback

How to install

  1. Click Download latest beta.
  2. Download the Windows .exe or .zip from the latest release.
  3. If Windows SmartScreen warns you, choose More info and then Run anyway only if you trust the source.
  4. Open the app and test with demo shots or your own match data.
